Part A — Composite Evaluation Using Reference Triangles
Problem 1
Find the exact value of \(\cos\!\left(\sin^{-1}\!\left(\dfrac{3}{5}\right)\right)\).
Reference Triangle Pythagorean Theorem
Let \(\theta = \sin^{-1}\!\left(\dfrac{3}{5}\right)\), so \(\sin\theta = \dfrac{3}{5}\).
Build right triangle: opp \(= 3\), hyp \(= 5\) → adj \(= \sqrt{25 - 9} = \sqrt{16} = 4\).
\(\cos\theta = \dfrac{\text{adj}}{\text{hyp}} = \dfrac{4}{5}\).
\(\dfrac{4}{5}\)
Problem 2
Find the exact value of \(\sin\!\left(\cos^{-1}\!\left(\dfrac{2}{3}\right)\right)\).
Reference Triangle
Let \(\theta = \cos^{-1}\!\left(\dfrac{2}{3}\right)\), so \(\cos\theta = \dfrac{2}{3}\).
Triangle: adj \(= 2\), hyp \(= 3\) → opp \(= \sqrt{9 - 4} = \sqrt{5}\).
\(\sin\theta = \dfrac{\sqrt{5}}{3}\).
\(\dfrac{\sqrt{5}}{3}\)
Problem 3
Find the exact value of \(\tan\!\left(\sin^{-1}\!\left(\dfrac{3}{5}\right)\right)\).
Reference Triangle
Same triangle as Problem 1: opp \(= 3\), adj \(= 4\), hyp \(= 5\).
\(\tan\theta = \dfrac{\text{opp}}{\text{adj}} = \dfrac{3}{4}\).
\(\dfrac{3}{4}\)
Problem 4
Find the exact value of \(\cos\!\left(\tan^{-1}\!\left(\dfrac{4}{3}\right)\right)\).
Reference Triangle
Let \(\theta = \tan^{-1}\!\left(\dfrac{4}{3}\right)\), so \(\tan\theta = \dfrac{4}{3}\).
Triangle: opp \(= 4\), adj \(= 3\) → hyp \(= \sqrt{16 + 9} = 5\).
\(\cos\theta = \dfrac{3}{5}\).
\(\dfrac{3}{5}\)
Problem 5
Write a simplified expression for \(\sin\!\left(\tan^{-1}\!\left(\dfrac{x}{2}\right)\right)\) in terms of \(x\).
Variable Triangle
Let \(\theta = \tan^{-1}\!\left(\dfrac{x}{2}\right)\), so \(\tan\theta = \dfrac{x}{2}\).
Triangle: opp \(= x\), adj \(= 2\) → hyp \(= \sqrt{x^2 + 4}\).
\(\sin\theta = \dfrac{x}{\sqrt{x^2 + 4}}\).
\(\dfrac{x}{\sqrt{x^2 + 4}}\)
Problem 6
Find the exact value of \(\sec\!\left(\cos^{-1}\!\left(\dfrac{1}{4}\right)\right)\).
Reciprocal Trig
Let \(\theta = \cos^{-1}\!\left(\dfrac{1}{4}\right)\), so \(\cos\theta = \dfrac{1}{4}\).
\(\sec\theta = \dfrac{1}{\cos\theta} = \dfrac{1}{\,1/4\,} = 4\).
\(4\)
Part B — Range-Aware Reduction
Problem 7
Evaluate \(\sin^{-1}\!\left(\sin\!\left(\dfrac{\pi}{3}\right)\right)\).
Range Check
Range of \(\sin^{-1}\) is \([-\tfrac{\pi}{2}, \tfrac{\pi}{2}]\).
\(\dfrac{\pi}{3} \approx 1.047 \in [-1.571, 1.571]\) → inside range.
Therefore \(\sin^{-1}(\sin(\tfrac{\pi}{3})) = \dfrac{\pi}{3}\).
\(\dfrac{\pi}{3}\)
Problem 8
Evaluate \(\sin^{-1}\!\left(\sin\!\left(\dfrac{2\pi}{3}\right)\right)\).
Reduction Required Range Check
\(\dfrac{2\pi}{3} \approx 2.094\). Range of \(\sin^{-1}\) is \([-\tfrac{\pi}{2}, \tfrac{\pi}{2}] \approx [-1.571, 1.571]\). → outside range.
Find angle in \([-\tfrac{\pi}{2}, \tfrac{\pi}{2}]\) with the same sine value: \(\sin\!\left(\tfrac{2\pi}{3}\right) = \sin\!\left(\pi - \tfrac{2\pi}{3}\right) = \sin\!\left(\tfrac{\pi}{3}\right) = \dfrac{\sqrt{3}}{2}\).
So \(\sin^{-1}\!\left(\dfrac{\sqrt{3}}{2}\right) = \dfrac{\pi}{3}\).
\(\dfrac{\pi}{3}\)
Problem 9
Evaluate \(\cos^{-1}\!\left(\cos\!\left(\dfrac{7\pi}{6}\right)\right)\).
Reduction Required Cosine Symmetry
Range of \(\cos^{-1}\) is \([0, \pi]\). But \(\dfrac{7\pi}{6} \approx 3.665 > \pi\) → outside range.
Find value: \(\cos\!\left(\dfrac{7\pi}{6}\right) = -\cos\!\left(\dfrac{\pi}{6}\right) = -\dfrac{\sqrt{3}}{2}\).
\(\cos^{-1}\!\left(-\dfrac{\sqrt{3}}{2}\right) = \pi - \dfrac{\pi}{6} = \dfrac{5\pi}{6}\).
\(\dfrac{5\pi}{6}\)
Problem 10
Evaluate \(\tan^{-1}\!\left(\tan\!\left(\pi\right)\right)\).
Reduction Required Tangent Period
Range of \(\tan^{-1}\) is \((-\tfrac{\pi}{2}, \tfrac{\pi}{2})\). \(\pi\) is outside this range.
\(\tan(\pi) = 0\).
\(\tan^{-1}(0) = 0\). (The angle inside the range with tangent value 0 is 0.)
\(0\)
Problem 11
Evaluate \(\sin^{-1}\!\left(\cos\!\left(\dfrac{3\pi}{2}\right)\right)\).
Co-function
First evaluate the inner function: \(\cos\!\left(\dfrac{3\pi}{2}\right) = 0\).
Then \(\sin^{-1}(0) = 0\), since \(0 \in [-\tfrac{\pi}{2}, \tfrac{\pi}{2}]\).
\(0\)
Part C — Double Angle & Advanced Identities
Problem 12
Find the exact value of \(\tan\!\left(2\sin^{-1}\!\left(\dfrac{1}{3}\right)\right)\).
Double Angle Reference Triangle
Let \(\theta = \sin^{-1}\!\left(\dfrac{1}{3}\right)\), so \(\sin\theta = \dfrac{1}{3}\).
Triangle: opp \(= 1\), hyp \(= 3\) → adj \(= \sqrt{9-1} = \sqrt{8} = 2\sqrt{2}\).
\(\tan\theta = \dfrac{1}{2\sqrt{2}}\).
Apply double-angle formula: \(\tan(2\theta) = \dfrac{2\tan\theta}{1 - \tan^2\theta}\).
\(\tan^2\theta = \dfrac{1}{8}\), so \(1 - \tan^2\theta = \dfrac{7}{8}\).
\(\tan(2\theta) = \dfrac{2 \cdot \tfrac{1}{2\sqrt{2}}}{\tfrac{7}{8}} = \dfrac{\tfrac{1}{\sqrt{2}}}{\tfrac{7}{8}} = \dfrac{1}{\sqrt{2}} \cdot \dfrac{8}{7} = \dfrac{8}{7\sqrt{2}} = \dfrac{8\sqrt{2}}{14} = \dfrac{4\sqrt{2}}{7}\).
\(\dfrac{4\sqrt{2}}{7}\)
Problem 13
Find the exact value of \(\cos\!\left(2\tan^{-1}\!\left(\dfrac{1}{2}\right)\right)\).
Double Angle Tangent Form
Let \(\theta = \tan^{-1}\!\left(\dfrac{1}{2}\right)\), so \(\tan\theta = \dfrac{1}{2}\).
Use: \(\cos(2\theta) = \dfrac{1 - \tan^2\theta}{1 + \tan^2\theta}\).
\(\tan^2\theta = \dfrac{1}{4}\).
\(\cos(2\theta) = \dfrac{1 - \tfrac{1}{4}}{1 + \tfrac{1}{4}} = \dfrac{\tfrac{3}{4}}{\tfrac{5}{4}} = \dfrac{3}{5}\).
\(\dfrac{3}{5}\)
Problem 14
Find the exact value of \(\sin\!\left(\sin^{-1}(0.8) + \cos^{-1}(0.6)\right)\).
Sum Formula Reference Triangles
Let \(\alpha = \sin^{-1}(0.8)\): \(\sin\alpha = 0.8 = \dfrac{4}{5}\), \(\cos\alpha = \dfrac{3}{5} = 0.6\).
Let \(\beta = \cos^{-1}(0.6)\): \(\cos\beta = 0.6 = \dfrac{3}{5}\), \(\sin\beta = \dfrac{4}{5} = 0.8\).
Apply \(\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta\):
\(= (0.8)(0.6) + (0.6)(0.8) = 0.48 + 0.48 = 0.96\).
\(0.96\)
Problem 15
Prove that \(\sin^{-1}(x) + \cos^{-1}(x) = \dfrac{\pi}{2}\) for all \(x \in [-1, 1]\).
Identity Proof Co-function
Proof:
Let \(\theta = \sin^{-1}(x)\), where \(\theta \in \left[-\dfrac{\pi}{2}, \dfrac{\pi}{2}\right]\).
Then \(\sin\theta = x\), which means \(x = \cos\!\left(\dfrac{\pi}{2} - \theta\right)\) (co-function identity).
So \(\cos^{-1}(x) = \cos^{-1}\!\left(\cos\!\left(\dfrac{\pi}{2} - \theta\right)\right)\).
We need \(\dfrac{\pi}{2} - \theta \in [0, \pi]\) for this to hold directly. Since \(\theta \in [-\tfrac{\pi}{2}, \tfrac{\pi}{2}]\), we have \(\dfrac{\pi}{2} - \theta \in [0, \pi]\). ✓ This is inside the range of \(\cos^{-1}\).
Therefore \(\cos^{-1}(x) = \dfrac{\pi}{2} - \theta = \dfrac{\pi}{2} - \sin^{-1}(x)\).
Rearranging: \(\sin^{-1}(x) + \cos^{-1}(x) = \dfrac{\pi}{2}\).   \(\blacksquare\)
\(\sin^{-1}(x) + \cos^{-1}(x) = \dfrac{\pi}{2}\)
